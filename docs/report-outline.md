# Weekly Report Outline

Quill turns market data into memo-style reports. The outline matters because the reader should be able to audit the thesis quickly.

## Suggested flow

1. Lead with the top market observation in one sentence.
2. Show the evidence that changed this week rather than repeating background.
3. Separate confirmed facts from interpretation.
4. End with the open question that deserves follow-up next week.

## Style rule

Short, sourced paragraphs beat long summaries. The report should read like a decision memo, not a blog post.
